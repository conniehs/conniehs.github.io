🚀 三步快速开始
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

第一步: 打开应用
   → 双击 index.html 文件
   → 或在浏览器中打开它
   
第二步: 允许麦克风
   → 浏览器会请求麦克风权限
   → 点击"允许" (Allow)
   
第三步: 开始使用
   → 点击"Start New Conversation"
   → 上传脚本或粘贴文本
   → 点击麦克风按钮录音
   → 查看分析结果

🎯 完整使用流程
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 首页 (Homepage)
   → 欢迎界面，选择"开始新对话"或"查看录音库"

2. 脚本上传 (Script Upload)
   → 填写脚本标题
   → 选择难度等级 (Beginner/Intermediate/Advanced)
   → 输入角色 (例如: Student A, Student B)
   → 上传.txt文件 或 粘贴对话内容
   → 点击"Save Script & Continue"

3. 录音工作室 (Recording Studio)
   → 点击大型麦克风按钮开始录音
   → 说出对话内容（根据脚本）
   → 观察实时波形和计时器
   → 再次点击停止录音
   → 使用播放器回听录音
   → 点击"Analyze Pronunciation"分析

4. 发音分析 (Analysis)
   → 查看Student A和Student B的分数
   → 查看综合得分
   → 查看前5个发音错误的单词
   → 点击单词听正确发音（UK English）
   → 学习和改进

5. 录音库 (Library)
   → 查看所有历史录音
   → 比较不同时期的进步
   → 删除不需要的录音

🔧 故障排除
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

问题: 麦克风不工作
解决: 检查浏览器地址栏的权限图标，允许麦克风访问

问题: 语音识别不准确
解决: 在安静环境录音，说话清晰，使用Chrome浏览器

问题: 无法听到TTS发音
解决: 检查系统音量，刷新页面重试

问题: 页面无法打开
解决: 确保使用现代浏览器（Chrome/Edge推荐）
